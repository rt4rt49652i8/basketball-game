# Basketball Game 2K26

A single-file Python basketball game inspired by NBA 2K26, built with Pygame.

## Quick Start

```bash
pip install -r requirements.txt
python basketball_2k26.py
```

## Controls

- **WASD** - Move player
- **SPACE** - Jump
- **ESC** - Pause/Resume
- **M** - Return to menu
- **Mouse Click** - Select game mode

## Features

✅ 2D Basketball Court with baskets
✅ Controllable players with physics
✅ Ball physics with gravity and collisions
✅ Real-time scoring
✅ Game HUD with score and timer
✅ Multiple game modes
✅ Pause menu

## Game Mechanics

- Move around the court with WASD
- Press SPACE to jump and reach higher positions
- Collide with the ball to bounce it towards baskets
- Score 2 points for each basket
- Try to outscore the other team!

## All in One File

Everything is contained in `basketball_2k26.py` including:
- Button class
- Player class
- Ball class
- Court class
- Game manager
- Main loop

Just run it and start playing!
